@ CSS란?
-> cascading style sheets의 약자
-> 웹 문서의 전반적인 스타일을 미리 저장해둔 스타일시트
-> 기존 HTML은 다양하게 설계하고, 수시로 변경하는데 많은 제약이 따르기 때문에 이를 보안하기
   위해 만들어진 것
-> 문서 내용 작성과 꾸미는 부분을 분리하여 내용을 수정해도 디자인을 바꿀 필요가 없고
   디자인을 수정해도 글 내용을 바꿀 필요가 없어짐
-> 다양한 기기에도 디자인만 따로 적용하고 구동 가능

@ style과 stylesheet
-> style은 정해진 속성을 입력하여 웹 페이지를 꾸미는 것
-> stylesheet는 웹페이지에서 반복적으로 쓰이는 style을 모아 놓은 것

# 스타일 기본형식

	p { text-align : center; }

	p		: 선택자
	text-align	: style 속성
	center		: 속성값

@ 내부 스타일 시트
-> <style></style> 내부에 스타일 정보를 저장하는 방법
-> 내부에 스타일 시트를 정의해놓고 사용하게 됨
(해당 페이지에서 계속 사용)

@ 외부 스타일 시트
-> <link> 태그를 이용하여 css파일을 읽어와서 스타일을 적용
-> 외부에서 가져온 것이기 때문에 다른 페이지에서도 호출해서 사용 가능

@ 인라인 스타일 시트
-> 태그 내부에 스타일 정보를 저장하는 방법
-> 해당 내부 태그에 정의한 것이기 때문에 해당 태그만 적용

@ CSS_선택자

선택자의 종류
전체 선택자 : *
태그 선택자 : 태그(h1,p,li 등)
아이디 선택자 : #아이디명
클래스 선택자 : .클래스명
후손 선택자 : 선택자 선택자
자손 선택자 : 선택자 > 선택자

속성 선택자 :   선택자[속성=값] / 선택자[속성~=값]  // (띄어쓰기로 구분되어 있어야함)
		선택자[속성|=값] / 선택자[속성^=값] // (-데쉬로 구분되어 있어야함)
		선택자[속성$=값] / 선택자[속성*=값] // (^ : 문자열로 시작하는거를 찾는데,
							중복은 안됨)
							($ : 문자열로 끝나는거)
							(* : 특정 문자를 포함하고 있는거)

동위 선택자 : 선택자 + 선택자 / 선택자 ~ 선택자

구조 선택자 :   선택자 : first-child / 선택자 : last-child
		선택자 : nth-child(수열)
		선택자 : nth-last-child(수열)
		선택자 : first-of-type / 선택자 : last-of-type
		선택자 : nth-of-type(수열)
		선택자 : nth-last-of-type(수열)

반응 선택자 : 선택자 : active / 선택자 : hover

상태 선택자 : :checked / :focus
속성 선택자 : :enabled / :disabled
링크 선택자 : :link / :visited
문자 선택자 :   ::first-letter / ::first-line
		::after / ::before / ::selection 
부정 선택자 : 선택자:not(선택자)

*ID는 하나의 고유한 것을 명시할 때 사용 (중복 불가능)
class는 그룹으로 묶어진 것을 명시할 때 사용 (중복 가능)

# CSS 기본속성
- CSS 텍스트 스타일
- CSS 문단 스타일
- CSS 목록&링크 스타일

@ CSS 단위 구성
상대 크기
     -> em  : 부모 요소의 크기가 기준 (배수를 의미)
     -> rem : 최상위의 부모 요소의 크기가 기준임
     -> %   : 기본 설정된 크기에서 상대적으로 크기 설정 // (브라우저의 몇배인지..)
절대 크기
     -> px  : 기본 지정된 크기
     -> pt  : 기본 지정된 크기 (포인트)

@ 색상표현
영문색 이름 : 영문으로 색이름을 작성
	red, blue, black 등
16진수 표현 : RGB값을 기준으로 16진수로 작성
	#ff0000 (빨간색) == #f00
rgb/rgba    : RGB값을 0~255로 입력, RGBA는 투명도 표현 가능
	- rgb(255,0,0)
	- rgb(0~255, 0~255, 0~255, 0~1) 입력
	0은 투명, 1은 불투명

(여기는 셤범위 아님 (밑))
@ 글꼴 속성
1. font-family
-> 폰트의 글꼴을 설정해주는 속성
-> 글꼴1이 없으면, 글꼴2, 글꼴3으로 선택되어 설정
-> 다 없으면 브라우저 기본 글꼴로 적용

2. font-size
- 글자의 크기를 조절하는 속성
단위 : em, px, pt, ex(안씀)

3. font-weight
- 글자 굵기를 조절하는 속성

구분값
normal	: 기본형태
bold	: 굵게 표시
bolder	: 기본보다 더 굵게 표시
lighter	: 기본보다 더 가늘게 표시
100~900	: 400이 normal, 700이 bold에 해당 (세밀한 조절 가능함)

4. font-variant
- 소문자 영어를 작은 대문자로 표시해주는 속성

5. font-style
-> 글자를 이텔릭체로 표시하는 속성

6. font
- 글꼴 속성을 모아서 표현할 수 있는 스타일 속성
- font-size와 font-family는 꼭 적어야 함 (나머지는 생략가능)
형식
- 선택자 { font : font-style font-variant
	font-weight font size/line-height
	font-family; } 형식

7. color
-> 글자색을 정하는 속성
-> rgb 색상표를 찾아서 해당 색 값을 적어서 사용

8. text-decoration
-> 글자에 밑줄을 긋거나, 취소선을 긋거나, 윗 선을 긋거나 밑줄을 표시하지 않는 속성
none	: 밑줄 삭제
underline : 밑줄 표시
overline : 윗줄 표시
line-through : 취소선 표시

9. text-transform
- 영문자를 표시할 때 대소문자를 원하는 대로 바꿀 수 있는 속성
none	: 변환없이 표시
capitalize : 시작하는 첫번째 글자를 대문자로 변환
uppercase : 모든 글자를 대문자로 반환
lowercase : 모든 글자를 소문자로 변환

10. text-shadow
- 텍스트에 그림자 효과를 주는 속성
none : 그림자 표현 안함
가로값 세로값 번짐값 색상값
- 가로 : 그림자가 가로로 얼마나 떨어져 있는지를 지정함 (양수는 오른쪽 음수는 왼쪽)
- 세로 : 그림자가 세로로 얼마나 떨어져 있는지를 지정함 (양수는 글자아래 음수는 글자 위)
- 번짐 : 그림자의 번지는 정도를 지정
- 색상 : 그림자의 색상을 지정함
(16진수, rgb 또는 색상이름 지정 가능)

11. white-space
- 공백을 처리해주는 속성
normal : 여러 개의 공백을 하나로 표시
nowrap : 여러 개 공백 한 개, 표시줄 바꾸지 않고 한줄 표시
pre : 여러 개 공백 그대로 표시 한줄로 표시
pre-line : 여러개 공백 한개 표시
영역 넘어가면 자동 줄 바꿈
pre-wrap : 여러 개 공백 그대로 표시
영역 넘어가면 자동 줄 바꿈

12. letter-spacing
- 낱개 글자의 간격을 조정하는 속성

13. word-spacing
- 단어와 단어 사이 간격을 조정하는 속성
